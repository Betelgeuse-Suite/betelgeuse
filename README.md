# beetlejuice
An all-in-one solution to manage configuration and language in native mobile apps.

## Global Dependencies

`npm i -g typescript`

`npm i -g typings`

`npm i -g ts-node`


## Usage

- Get an output of the next concatonated build

`ts-node ./src/index.ts --prev ./mocks/a|prev|*.json --src ./mocks/lang`
